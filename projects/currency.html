CURRENCY CONVERTER

AngularJS App

Background

This web app was inspired by mobile currency apps. I decided to create a web version that mimicked some of the functionality and style of these apps.

Technical info
  • This is an SPA(Single Page Application) built using the AngularJS JavaScript framework
  • The currency data is acquired through an API.
  • The design was done with Bootstrap.
  • This is a mobile friendly app
