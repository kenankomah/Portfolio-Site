SHOOTER

HTML5 & JavaScript Game

Background

This game is a variation of old arcade games such as Asteroids.

Technical info
  • The animation and programming was all done with hand coded JavaScript.
  • I used the HTML5 canvas element to create the graphics for the game.
  • For the sound I used the JavaScript audio object.
  • The explosion effects were done using sprites.
