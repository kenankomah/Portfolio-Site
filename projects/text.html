TEXT ANALYSER

HTML & JavaScript App

Background

This is a simple app which creates a breakdown of words typed or pasted into the text box along with the frequency of each word.

Technical info
  • I used JavaScript to create the word breakdown and frequency count.
  • The two keyword analysis are done using an API.
  • To get the keyword analysis the text is URI encoded so it can be analysed using the API.
